# Nigeria Healthcare 2024 Dashboard
This Power BI project analyzes healthcare trends in Nigeria using interactive dashboards and data visualization techniques.
## Project Overview
The dashboard provides insights into:
* Patient admissions
* Hospital revenue
* Average treatment costs
* Mortality rates
* Readmission trends
* Monthly admission patterns
* Payment methods
* Diagnosis distribution
## Key Insights
* Malaria recorded the highest number of diagnosed cases
* Lagos and Port Harcourt generated the highest revenue
* The average patient stay was 8.09 days
* The mortality rate was 32.63%
## Tools Used
* Power BI
* Data Cleaning
* Data Visualization
* Dashboard Design
* Analytical Storytelling

## Project Goal
The goal of this project was to transform raw healthcare data into actionable insights that can support healthcare decision-making and resource planning.
